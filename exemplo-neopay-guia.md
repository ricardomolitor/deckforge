# Guia do Usuário — Sistema NeoPay 3.0

## Visão Geral
O NeoPay 3.0 é a plataforma de pagamentos digitais da próxima geração. Integra PIX, cartões, boletos e carteiras digitais em uma única API unificada.

## Principais Funcionalidades

### 1. Dashboard Unificado
- Visão 360° de todas as transações em tempo real
- Gráficos interativos de volume, ticket médio e taxa de conversão
- Alertas inteligentes de anomalias (powered by ML)

### 2. API Gateway
- REST + GraphQL com latência < 50ms
- Rate limiting adaptativo
- Sandbox completo para desenvolvimento

### 3. Motor Antifraude
- Score de risco em tempo real (0-1000)
- Regras customizáveis por merchant
- Machine Learning com 99.2% de acurácia
- Redução de 67% em chargebacks

### 4. Conciliação Automática
- Matching automático de recebíveis
- Suporte a split de pagamento
- Relatórios contábeis prontos para ERP

### 5. Portal do Merchant
- Onboarding self-service em 15 minutos
- KYC/KYB automatizado
- Customização de checkout (white-label)

## Métricas do Sistema
- 12 milhões de transações/dia
- Uptime 99.99% nos últimos 12 meses
- 2.400 merchants ativos
- NPS de 78 (vs. 45 da média do setor)
- Tempo médio de integração: 3 dias (vs. 30 dias concorrência)
